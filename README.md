# Tic Tac Toe 

A new Flutter project.

A Flutter Tic Tac Toe app with polished UI/UX, interactive animations, and smart game logic.

## Features
- Smart Game Logic: Win/draw detection and score tracking for Player X & O.
- Dynamic Theming: Light & Dark mode with adaptive colors.
- Interactive Cards: Pop + Scale + Tween animations on tap.
- Win Celebration: Confetti highlights winning combination.
- Localization: Supports English (LTR) and Arabic (RTL).

## Technical Stack
- Flutter & Dart
- AnimatedContainer, TweenAnimationBuilder
- Confetti package
- Responsive layout with LayoutBuilder
